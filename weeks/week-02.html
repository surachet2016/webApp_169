โครงสร้างพื้นฐานของ HTML

HTML (HyperText Markup Language) ใช้ Tags เพื่อกำหนดโครงสร้างเนื้อหา ทุกไฟล์ HTML ที่ถูกต้องต้องเริ่มด้วย <!DOCTYPE html> เพื่อบอกเบราว์เซอร์ว่า นี่คือเอกสาร HTML5 ตามด้วย <html>, <head>, และ <body>

HTML5 นำเสนอ Semantic Elements ซึ่งเป็น Tags ที่มีความหมายในตัวเอง ช่วยให้โค้ดอ่านง่าย และช่วย Search Engine ทำความเข้าใจโครงสร้างหน้าเว็บได้ดีขึ้น:

  • <header> — ส่วนหัวของหน้าหรือ section (Logo, Navigation)
  • <nav> — เมนูนำทาง
  • <main> — เนื้อหาหลักของหน้า (มีได้เพียง 1 อัน)
  • <section> — ส่วนของเนื้อหาที่มีหัวข้อ
  • <article> — เนื้อหาที่เป็นอิสระ เช่น บทความ โพสต์
  • <aside> — เนื้อหาเสริมข้างเคียง เช่น Sidebar
  • <footer> — ส่วนท้ายของหน้า

Elements แบ่งออกเป็น Block-level (ขึ้นบรรทัดใหม่เสมอ เช่น <div>, <p>, <h1>-<h6>) และ Inline (อยู่ในบรรทัดเดียวกัน เช่น <span>, <a>, <strong>, <img>)

เคล็ดลับ

ใช้ Semantic Elements แทน <div> ทั่วไปเสมอเมื่อมีความหมายที่ตรงกัน เช่น ใช้ <nav> แทน <div class="nav"> ช่วยให้ Screen Reader อ่านหน้าเว็บได้ถูกต้อง และ SEO ดีขึ้น

CSS: การจัดรูปแบบและ Layout

CSS (Cascading Style Sheets) ใช้ Selectors เพื่อเลือก Elements แล้วกำหนด Properties: element (เลือกทุก tag), .class (เลือกตาม class), #id (เลือก id เดียว), element:hover (Pseudo-class เมื่อเมาส์ชี้), parent > child (เลือก child โดยตรง)

Box Model คือแนวคิดพื้นฐานของ CSS Layout ทุก Element มี: Content → Padding → Border → Margin เรียงจากใน-ออก ใช้ box-sizing: border-box เพื่อให้ width/height รวม padding และ border ด้วย

Flexbox เหมาะสำหรับ Layout 1 มิติ (แถวหรือคอลัมน์):

  • display: flex — เปิดใช้ Flexbox บน container
  • flex-direction: row | column — ทิศทางการวาง items
  • justify-content — จัดตำแหน่งในแนวแกนหลัก (flex-start, center, space-between, space-around)
  • align-items — จัดตำแหน่งในแนวตั้งฉาก (flex-start, center, stretch)
  • flex-wrap: wrap — อนุญาตให้ items ตัดบรรทัดใหม่
  • gap — ระยะห่างระหว่าง items

CSS Grid เหมาะสำหรับ Layout 2 มิติ (แถวและคอลัมน์พร้อมกัน):

  • display: grid — เปิดใช้ Grid
  • grid-template-columns: repeat(3, 1fr) — 3 คอลัมน์เท่ากัน
  • grid-template-areas — ตั้งชื่อพื้นที่ใน Grid
  • grid-column: span 2 — ขยาย item ให้ครอบ 2 คอลัมน์

Responsive Design ด้วย @media query ให้เว็บปรับตัวตามขนาดหน้าจอ:

  • Mobile: @media (max-width: 576px)
  • Tablet: @media (max-width: 768px)
  • Desktop: @media (min-width: 992px)

JavaScript พื้นฐาน

JavaScript (JS) เป็นภาษาโปรแกรมที่รันบนเบราว์เซอร์ ทำให้หน้าเว็บมีพฤติกรรมแบบ Dynamic ใช้ let และ const แทน var เสมอ: const สำหรับค่าที่ไม่เปลี่ยนแปลง, let สำหรับค่าที่เปลี่ยนได้

DOM Manipulation คือการเข้าถึงและแก้ไข Elements ใน HTML ผ่าน JavaScript:

  • document.querySelector('.class') — เลือก Element แรกที่ตรง
  • document.querySelectorAll('li') — เลือกทุก Element ที่ตรง (NodeList)
  • element.textContent = 'text' — เปลี่ยนข้อความ
  • element.innerHTML = '<b>bold</b>' — เปลี่ยน HTML ภายใน
  • element.classList.add('active') — เพิ่ม class
  • element.classList.toggle('hidden') — toggle class
  • element.style.color = 'red' — เปลี่ยน Inline Style

Event Listeners ใช้รับเหตุการณ์จากผู้ใช้ เช่น การคลิก การพิมพ์ การ Submit ฟอร์ม:

  • element.addEventListener('click', handler)
  • element.addEventListener('submit', handler)
  • element.addEventListener('input', handler)
  • event.preventDefault() — ป้องกันพฤติกรรมเริ่มต้น (เช่น ไม่ให้ฟอร์ม Submit)

ข้อควรระวัง

อย่าใช้ innerHTML กับข้อมูลที่มาจากผู้ใช้หรือ API โดยตรง เพราะอาจเกิดช่องโหว่ XSS (Cross-Site Scripting) ได้ ให้ใช้ textContent แทนเมื่อต้องการแสดงข้อความ

การจัดวาง Layout หน้าเว็บ

Layout ที่พบบ่อยที่สุดในเว็บแอปพลิเคชัน คือ Header + Sidebar + Main Content + Footer สามารถสร้างด้วย CSS Grid template areas ได้อย่างสวยงาม: 

┌─────────────────────────────────────────┐
│              HEADER                     │  grid-area: header
├──────────┬──────────────────────────────┤
│          │                              │
│ SIDEBAR  │     MAIN CONTENT             │  grid-area: sidebar / main
│          │                              │
│          │                              │
├──────────┴──────────────────────────────┤
│              FOOTER                     │  grid-area: footer
└─────────────────────────────────────────┘

CSS Grid Template Areas:
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";

บน Mobile (หน้าจอเล็ก) ให้เปลี่ยน Layout เป็น 1 คอลัมน์ด้วย @media query: Sidebar จะซ้อนอยู่ใต้ Header และ Main Content จะอยู่ถัดลงมา

form-validation.js JavaScript 

// ============================
// Form Validation ด้วย JavaScript
// ============================

const form = document.getElementById('registerForm');
const nameInput = document.getElementById('name');
const emailInput = document.getElementById('email');
const phoneInput = document.getElementById('phone');

// ฟังก์ชันแสดงข้อผิดพลาดใต้ input
function showError(input, message) {
  const field = input.parentElement;
  // ลบ error เก่าออกก่อน
  const existingError = field.querySelector('.error-msg');
  if (existingError) existingError.remove();

  // เพิ่ม error ใหม่
  const errorDiv = document.createElement('div');
  errorDiv.className = 'error-msg';
  errorDiv.textContent = message;
  field.appendChild(errorDiv);
  input.classList.add('input-error');
}

// ฟังก์ชันล้าง error
function clearError(input) {
  const field = input.parentElement;
  const existingError = field.querySelector('.error-msg');
  if (existingError) existingError.remove();
  input.classList.remove('input-error');
}

// Validate email format
function isValidEmail(email) {
  const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return regex.test(email);
}

// Validate Thai phone number (0xx-xxx-xxxx)
function isValidPhone(phone) {
  const regex = /^0[0-9]{8,9}$/;
  return regex.test(phone.replace(/-/g, ''));
}

// Event: ตรวจสอบเมื่อ Submit
form.addEventListener('submit', function(event) {
  event.preventDefault(); // ป้องกันการ Submit จริง

  let isValid = true;

  // ตรวจสอบชื่อ
  if (nameInput.value.trim() === '') {
    showError(nameInput, 'กรุณากรอกชื่อ-นามสกุล');
    isValid = false;
  } else if (nameInput.value.trim().length < 3) {
    showError(nameInput, 'ชื่อต้องมีอย่างน้อย 3 ตัวอักษร');
    isValid = false;
  } else {
    clearError(nameInput);
  }

  // ตรวจสอบ Email
  if (emailInput.value.trim() === '') {
    showError(emailInput, 'กรุณากรอก Email');
    isValid = false;
  } else if (!isValidEmail(emailInput.value)) {
    showError(emailInput, 'รูปแบบ Email ไม่ถูกต้อง');
    isValid = false;
  } else {
    clearError(emailInput);
  }

  // ตรวจสอบเบอร์โทร
  if (phoneInput.value.trim() === '') {
    showError(phoneInput, 'กรุณากรอกเบอร์โทรศัพท์');
    isValid = false;
  } else if (!isValidPhone(phoneInput.value)) {
    showError(phoneInput, 'เบอร์โทรต้องเป็นตัวเลข 9-10 หลัก');
    isValid = false;
  } else {
    clearError(phoneInput);
  }

  if (isValid) {
    console.log('Form ผ่านการตรวจสอบ! พร้อม Submit');
    // ส่งข้อมูลไปยัง Server ได้เลย
    form.submit();
  }
});

// Event: ล้าง error เมื่อผู้ใช้เริ่มพิมพ์
[nameInput, emailInput, phoneInput].forEach(input => {
  input.addEventListener('input', () => clearError(input));
});

// ฟังก์ชันสร้าง Card HTML จากข้อมูลนักศึกษา
function createStudentCard(student) {
  const gpaColor = student.gpa >= 3.5 ? '#27ae60' : student.gpa >= 3.0 ? '#f39c12' : '#e74c3c';

  const card = document.createElement('div');
  card.className = 'student-card';
  card.innerHTML = `
    
${student.name.charAt(0)}

    

      
${student.name}

      
รหัส: ${student.id}

      
คณะ: ${student.faculty}

      GPA: ${student.gpa.toFixed(2)}
    

  `;
  return card;
}

// ฟังก์ชันแสดงผลในหน้าเว็บ
function renderStudents(data) {
  const container = document.getElementById('studentContainer');
  container.innerHTML = ''; // ล้างข้อมูลเก่า

  if (data.length === 0) {
    container.innerHTML = '
ไม่พบข้อมูลนักศึกษา
';
    return;
  }

  data.forEach(student => {
    container.appendChild(createStudentCard(student));
  });
}

// ตัวอย่าง Fetch จาก API จริง
// ============================
async function fetchFromAPI() {
  const loadingEl = document.getElementById('loading');
  loadingEl.style.display = 'block';

  try {
    const response = await fetch('https://jsonplaceholder.typicode.com/users');

    // ตรวจสอบว่า Response สำเร็จ
    if (!response.ok) {
      throw new Error(`HTTP Error: ${response.status}`);
    }

    const users = await response.json();
    console.log('ดึงข้อมูลสำเร็จ:', users.length, 'รายการ');

    // แปลงข้อมูลให้อยู่ในรูปแบบที่ต้องการ
    const formattedUsers = users.map(user => ({
      id: String(user.id).padStart(7, '0'),
      name: user.name,
      faculty: user.company.name,
      gpa: (Math.random() * 1.5 + 2.5).toFixed(2) * 1
    }));

    renderStudents(formattedUsers);

  } catch (error) {
    console.error('เกิดข้อผิดพลาด:', error.message);
    document.getElementById('studentContainer').innerHTML =
      `
ไม่สามารถดึงข้อมูลได้: ${error.message}
`;
  } finally {
    loadingEl.style.display = 'none';
  }
}

renderStudents(students);

🧪 ปฏิบัติการ Lab 1 — ทบทวน HTML, CSS, JavaScript

ฝึกสร้างหน้าเว็บด้วย HTML Semantic, จัด Layout ด้วย CSS Flexbox/Grid, และเพิ่ม Interactivity ด้วย JavaScript

1
สร้างหน้าเว็บส่วนตัว (Personal Profile Page)

สร้างไฟล์ profile.html ที่แสดงประวัติส่วนตัวของคุณ โดยใช้ Semantic HTML5 Elements ให้ถูกต้อง

  • ใช้ <header> แสดงชื่อและรูปภาพ (ใช้ placeholder จาก https://ui-avatars.com/api/?name=ชื่อคุณ)
  • ใช้ <main> และ <section> แบ่งส่วน: ประวัติ, ทักษะ, การศึกษา
  • ใช้ <ul> หรือ <ol> แสดงรายการทักษะอย่างน้อย 5 ข้อ
  • ใช้ <footer> แสดงข้อมูลติดต่อ (Email, เบอร์โทร)
  • ใช้ <time> สำหรับวันที่เกิดหรือวันที่เข้าศึกษา
เกณฑ์การผ่าน

หน้าเว็บแสดงข้อมูลครบ 4 ส่วน (header, main-bio, main-skills, footer) โดยใช้ Semantic Elements ที่ถูกต้อง ตรวจสอบด้วย W3C Validator (validator.w3.org) ต้องไม่มี Error

คำแนะนำ

เปิด Chrome DevTools กด Elements tab แล้วดู Tree ของ HTML ตรวจสอบว่า Elements ซ้อนกันถูกต้อง เช่น <ul> ต้องมี <li> เป็น Child โดยตรง

2
ออกแบบ Layout ด้วย CSS Flexbox

สร้างไฟล์ cards.html และ cards.css โดยใช้ CSS Flexbox จัด Layout การ์ด 3 คอลัมน์ที่ปรับเป็น 1 คอลัมน์บน Mobile

  • สร้างการ์ดอย่างน้อย 6 ใบ แต่ละใบมี: รูปภาพ, หัวข้อ, รายละเอียด, ปุ่ม
  • ใช้ display: flex; flex-wrap: wrap; บน container
  • กำหนด flex: 0 0 calc(33.33% - 16px) หรือใช้ CSS Grid repeat(3, 1fr)
  • เพิ่ม Hover Effect บนการ์ด: เงา (box-shadow) และขยับขึ้น (transform: translateY)
  • ใช้ @media (max-width: 768px) เปลี่ยนเป็น 1 คอลัมน์
เกณฑ์การผ่าน

บน Desktop (>768px) การ์ดแสดง 3 ใบต่อแถว และบน Mobile (<=768px) แสดง 1 ใบต่อแถว ทดสอบโดยใช้ DevTools Responsive Mode กดปุ่มไอคอน "Toggle device toolbar" (Ctrl+Shift+M)

คำแนะนำ

หากใช้ CSS Grid ให้ลอง grid-template-columns: repeat(auto-fill, minmax(280px, 1fr)) ซึ่งจะปรับจำนวนคอลัมน์อัตโนมัติตามขนาดหน้าจอโดยไม่ต้องเขียน @media

3
สร้าง Navigation Bar แบบ Responsive

สร้าง Navigation Bar ที่มีเมนูปกติบน Desktop และเปลี่ยนเป็น Hamburger Menu บน Mobile

  • สร้าง <nav> ด้วย Logo ด้านซ้าย และรายการเมนู 5 รายการด้านขวา
  • เพิ่มปุ่ม Hamburger (☰) ที่ซ่อนอยู่บน Desktop แต่แสดงบน Mobile
  • ใช้ JavaScript addEventListener('click', ...) ให้ปุ่ม Hamburger toggle class active
  • ใช้ CSS ซ่อน/แสดงเมนูด้วย class active บน Mobile
  • เพิ่ม Active State: Highlight เมนูที่กำลังดูอยู่ด้วย CSS
เกณฑ์การผ่าน

บน Desktop เมนูแสดงแนวนอน, บน Mobile (<=768px) มีปุ่ม Hamburger และกดแล้วเมนูเปิด/ปิดได้ เมนูทุกลิงก์ต้องคลิกได้และมี Hover Effect

คำแนะนำ

วิธีง่ายที่สุดคือ toggle class บน <nav> แล้วใช้ CSS: nav .menu { display: none; } และ nav.active .menu { display: flex; flex-direction: column; }

JavaScript: ตรวจสอบแบบฟอร์ม (Form Validation)

สร้างฟอร์มลงทะเบียนใน register.html พร้อม Validation แบบ Real-time ด้วย JavaScript

  • ฟอร์มมีช่องกรอก: ชื่อ-นามสกุล, Email, เบอร์โทรศัพท์, รหัสผ่าน, ยืนยันรหัสผ่าน
  • ตรวจสอบว่าไม่ว่างเปล่า, Email format ถูกต้อง, เบอร์โทรเป็น 10 หลัก, รหัสผ่านตรงกัน
  • แสดง Error Message ใต้แต่ละ field ที่มีปัญหา (สีแดง)
  • เมื่อผู้ใช้แก้ไขให้ Error หายไปทันที (ใช้ event input)
  • ปุ่ม Submit ให้แสดง Alert ว่า "ลงทะเบียนสำเร็จ!" เมื่อผ่านทุกการตรวจสอบ
เกณฑ์การผ่าน

ทดสอบกรอกข้อมูลผิดทุก field และกด Submit — ต้องเห็น Error messages ทุกช่อง จากนั้นแก้ไขทีละช่อง Error ต้องหายไปทีละอัน และเมื่อกรอกถูกทั้งหมดกด Submit แล้วได้ Alert "ลงทะเบียนสำเร็จ!"

คำแนะนำ

ใช้ Regular Expression สำหรับ Email: /^[^\s@]+@[^\s@]+\.[^\s@]+$/ และสำหรับเบอร์โทรไทย: /^0[0-9]{9}$/ ทดสอบด้วย regex.test(value)

5
JavaScript: แสดงข้อมูลแบบ Dynamic จาก Array

สร้างไฟล์ students.html ที่แสดงรายชื่อนักศึกษาเป็น Cards แบบ Dynamic โดยดึงจาก Array ใน JavaScript

  • สร้าง Array ของ Objects นักศึกษาอย่างน้อย 8 คน มี properties: name, studentId, faculty, gpa
  • เขียนฟังก์ชัน renderStudents(data) ที่สร้าง HTML การ์ดจาก Array และแทรกใน DOM
  • เพิ่มช่องค้นหา: กรอกชื่อแล้วกรอง Array และ render ใหม่
  • เพิ่มปุ่ม Sort: เรียงตาม GPA (มาก-น้อย) และ (น้อย-มาก)
  • แสดงจำนวนนักศึกษาที่พบ เช่น "พบ 8 รายการ"
เกณฑ์การผ่าน

การ์ดแสดงครบทุกคน, ค้นหาชื่อแล้วกรองได้ถูกต้อง (ไม่ Case-Sensitive), กด Sort แล้วลำดับเปลี่ยน และตัวนับแสดงจำนวนที่ถูกต้องตลอด

คำแนะนำ

ใช้ Array.filter() สำหรับค้นหาและ Array.sort() สำหรับเรียง ตัวอย่าง: students.filter(s => s.name.includes(searchText)) และ students.sort((a, b) => b.gpa - a.gpa)
