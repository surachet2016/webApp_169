PHP คืออะไร?

PHP (PHP: Hypertext Preprocessor) เป็นภาษา Scripting ฝั่ง Server (Server-side Scripting Language) ที่ถูกออกแบบมาสำหรับการพัฒนาเว็บโดยเฉพาะ PHP ทำงานบน Web Server (เช่น Apache) ก่อนที่ผลลัพธ์จะถูกส่งไปยังเบราว์เซอร์ ผู้ใช้จะเห็นแค่ HTML ที่ PHP สร้างขึ้น ไม่เห็นโค้ด PHP โดยตรง

จุดเด่นของ PHP ได้แก่:

  • ฝังใน HTML ได้ — เขียน PHP ปนกับ HTML ในไฟล์เดียวกัน ด้วย PHP Tags <?php ... ?>
  • Open Source & ฟรี — ดาวน์โหลดและใช้งานได้ฟรี ชุมชนผู้ใช้ขนาดใหญ่
  • รองรับ Database หลายชนิด — MySQL, PostgreSQL, SQLite, MongoDB
  • Cross-platform — ทำงานได้บน Windows, macOS, Linux
  • ใช้งานแพร่หลาย — WordPress, Facebook (เดิม), Wikipedia ล้วนสร้างด้วย PHP

ไฟล์ PHP จะมีนามสกุล .php และต้องรันผ่าน Web Server เท่านั้น (ไม่สามารถเปิดด้วยการดับเบิ้ลคลิกไฟล์แบบ HTML ได้) นั่นคือต้องเข้าถึงผ่าน URL เช่น http://localhost/myproject/index.php

การติดตั้ง XAMPP

XAMPP เป็นชุดซอฟต์แวร์ที่รวม Web Server และ Database ไว้ในชุดเดียว ประกอบด้วย:

  • X — Cross-platform (ทำงานได้บน Windows, macOS, Linux)
  • A — Apache Web Server (ให้บริการ HTTP)
  • M — MariaDB/MySQL (ระบบจัดการฐานข้อมูล)
  • P — PHP (ภาษา Scripting)
  • P — Perl (ภาษา Scripting อีกตัว)

ขั้นตอนการติดตั้งและใช้งาน XAMPP:

  • ดาวน์โหลดจาก https://www.apachefriends.org เลือก Version สำหรับ OS ของคุณ
  • ติดตั้งด้วยการรัน Installer ยอมรับ Default Settings
  • เปิด XAMPP Control Panel แล้วกด Start ที่ Apache และ MySQL
  • ตัวบ่งชี้สีเขียวแสดงว่า Service กำลังทำงาน
  • วางไฟล์ PHP ใน folder htdocs (Windows: C:\xampp\htdocs\, macOS: /Applications/XAMPP/htdocs/)
  • เข้าถึงได้ที่ http://localhost/ หรือ http://127.0.0.1/
  • เปิด phpMyAdmin (จัดการ MySQL) ที่ http://localhost/phpmyadmin
ข้อควรระวัง — Port Conflicts

Apache ใช้ Port 80 (HTTP) และ 443 (HTTPS) หากมีโปรแกรมอื่นใช้ Port เดียวกัน (เช่น Skype, IIS) Apache จะ Start ไม่ได้ แก้ไขโดย: หยุดโปรแกรมที่ใช้ Port ซ้อน หรือเปลี่ยน Port ของ Apache ใน httpd.conf เช่นเปลี่ยนจาก 80 เป็น 8080 แล้วเข้าถึงด้วย http://localhost:8080/

โครงสร้างภาษา PHP พื้นฐาน

PHP Tags บอกให้ PHP Engine รู้ว่าส่วนใดคือโค้ด PHP ที่ต้องประมวลผล:

  • <?php ... ?> — Standard PHP Tag (แนะนำ)
  • <?= $variable ?> — Short Echo Tag สำหรับแสดงค่าตัวแปร

ตัวแปร (Variables) ใน PHP เริ่มต้นด้วยเครื่องหมาย $ เสมอ PHP เป็น Loosely Typed Language ไม่ต้องประกาศชนิดข้อมูลล่วงหน้า:

  • $name = "สมชาย"; — String (ข้อความ)
  • $age = 20; — Integer (จำนวนเต็ม)
  • $gpa = 3.75; — Float (ทศนิยม)
  • $isStudent = true; — Boolean (จริง/เท็จ)
  • $scores = [85, 90, 78]; — Array (อาร์เรย์)
  • $nothing = null; — Null (ไม่มีค่า)

คำสั่งแสดงผล: echo และ print ใช้แสดงข้อความหรือค่าตัวแปร var_dump() ใช้แสดงชนิดและค่าของตัวแปร (มีประโยชน์สำหรับ Debug):

เคล็ดลับ — String Concatenation

PHP ใช้จุด (.) สำหรับต่อสตริง ไม่ใช่ + แบบ JavaScript เช่น: echo "สวัสดี " . $name . "!"; หรือฝังตัวแปรใน Double Quotes ได้เลย: echo "สวัสดี $name!";

PHP กับ HTML

จุดแข็งของ PHP คือสามารถฝังในไฟล์ HTML ได้โดยตรง ทำให้สร้างหน้าเว็บ Dynamic ได้ง่าย PHP จะประมวลผลส่วนที่อยู่ใน <?php ?> แล้วส่ง Output ออกมาแทนที่ Tags นั้น

คำสั่ง include และ require ใช้แทรกไฟล์ PHP อื่นเข้ามา เหมาะสำหรับแยก Header, Footer, หรือ Config เป็นไฟล์แยกและใช้ซ้ำในหลายหน้า:

  • include 'header.php'; — แทรกไฟล์ (หากไม่พบจะแสดง Warning และรันต่อ)
  • require 'config.php'; — แทรกไฟล์ (หากไม่พบจะหยุดทำงานทันที)
  • include_once / require_once — แทรกไฟล์เพียงครั้งเดียว (ป้องกัน Duplicate)

🧪 ปฏิบัติการ Lab 2 — PHP เบื้องต้น

ฝึกติดตั้ง XAMPP, สร้างไฟล์ PHP แรก, ทดลองชนิดข้อมูล, และสร้างหน้าเว็บ Dynamic ด้วย PHP

1
ติดตั้งและทดสอบ XAMPP

ติดตั้ง XAMPP และทดสอบว่า Apache และ MySQL ทำงานได้อย่างถูกต้อง

  • ดาวน์โหลด XAMPP จาก https://www.apachefriends.org
  • ติดตั้งและเปิด XAMPP Control Panel
  • กด Start Apache และ Start MySQL — ทั้งคู่ต้องเป็นสีเขียว
  • เปิดเบราว์เซอร์ไปที่ http://localhost — ต้องเห็นหน้า XAMPP Dashboard
  • เปิด http://localhost/phpmyadmin — ต้องเห็นหน้า phpMyAdmin
  • สร้างไฟล์ test.php ใน htdocs ด้วยเนื้อหา <?php echo "PHP ทำงานได้!"; ?>
  • เปิด http://localhost/test.php — ต้องเห็นข้อความ "PHP ทำงานได้!"
เกณฑ์การผ่าน

สามารถเปิด http://localhost เห็น XAMPP Dashboard, เปิด http://localhost/phpmyadmin ได้, และ http://localhost/test.php แสดงข้อความที่ถูกต้อง ถ่ายภาพหน้าจอทั้ง 3 หน้าส่งอาจารย์

คำแนะนำ

หาก Apache Start ไม่ได้ ให้ตรวจสอบว่ามีโปรแกรมอื่นใช้ Port 80 หรือไม่ เปิด Command Prompt แล้วพิมพ์ netstat -ano | findstr :80 (Windows) หรือ lsof -i :80 (macOS) เพื่อดูว่ามีโปรแกรมใดใช้ Port นั้น

2
สร้างไฟล์ PHP แรก

สร้างไฟล์ profile.php ใน htdocs ที่แสดงข้อมูลส่วนตัวของคุณพร้อมวันที่และเวลา

  • แสดงชื่อ-นามสกุล, รหัสนักศึกษา, คณะ, และสาขาวิชา
  • แสดงวันที่ปัจจุบันในรูปแบบ dd/mm/yyyy ด้วย date('d/m/Y')
  • แสดงเวลาปัจจุบันด้วย date('H:i:s')
  • คำนวณอายุจากปีเกิด: $age = date('Y') - $birthYear;
  • แสดง Output ในรูปแบบ HTML ที่สวยงาม (มี CSS ตกแต่ง)
เกณฑ์การผ่าน

เปิด http://localhost/profile.php แล้วเห็นข้อมูลครบทุกอย่าง วันที่และเวลาต้องแสดงค่าปัจจุบันจริง ๆ (ไม่ใช่ค่า Hardcode) และ HTML ต้องถูกต้อง ไม่มี PHP Tags รั่วออกมาแสดงในหน้า

คำแนะนำ

หากหน้าเว็บแสดง PHP Code โดยตรงแทนผลลัพธ์ แสดงว่า Apache ไม่ได้ Start หรือคุณเปิดไฟล์โดยตรง (ผ่าน file://) ต้องเข้าผ่าน http://localhost/... เสมอ

3
ทดลองชนิดข้อมูล PHP

สร้างไฟล์ datatypes.php ที่ทดลองตัวแปรทุกชนิดและแสดงผลด้วย var_dump()

  • ประกาศตัวแปร String, Integer, Float, Boolean, Array, Null อย่างน้อยชนิดละ 2 ตัว
  • แสดงผลด้วย var_dump() ทุกตัวแปรและบันทึกผลที่ได้
  • ทดลอง Type Casting: แปลง String "100" เป็น Integer, แปลง Float 3.9 เป็น Integer
  • ทดลอง Type Juggling: คำนวณ "5" + 3 และบันทึกผล (PHP แปลง String เป็น Integer อัตโนมัติ)
  • ทดลองฟังก์ชัน gettype() เพื่อดูชนิดข้อมูล
เกณฑ์การผ่าน

หน้าเว็บแสดง var_dump() ของตัวแปรทุกชนิดพร้อมชนิดข้อมูล เช่น string(5) "hello", int(20) นักศึกษาต้องอธิบายได้ว่าผลของ (int)3.9 คืออะไร (คำตอบ: 3 ตัดทศนิยมทิ้ง)

คำแนะนำ

ใช้ <pre> ครอบ var_dump() เพื่อให้ Output จัดรูปแบบสวยงาม: echo "<pre>"; var_dump($myArray); echo "</pre>";

4
สร้างหน้าเว็บด้วย PHP และ HTML

สร้างไฟล์ greeting.php ที่แสดงการทักทายตามเวลาของวันแบบ Dynamic

  • ดึงชั่วโมงปัจจุบันด้วย $hour = (int)date('H');
  • เงื่อนไข: 5-11 = "สวัสดีตอนเช้า ☀️", 12-17 = "สวัสดีตอนบ่าย 🌤️", 18-21 = "สวัสดีตอนเย็น 🌆", อื่น ๆ = "สวัสดียามค่ำคืน 🌙"
  • แสดงวันในสัปดาห์เป็นภาษาไทย: ใช้ date('N') (1=จันทร์, 7=อาทิตย์) พร้อม Array ['จันทร์','อังคาร','พุธ','พฤหัสฯ','ศุกร์','เสาร์','อาทิตย์']
  • นับวันถึงสิ้นปีและแสดง: $daysLeft = ceil((mktime(0,0,0,12,31,date('Y')) - time()) / 86400);
  • ออกแบบหน้าให้สวยงามด้วย CSS (สีพื้นหลังตามช่วงเวลา)
เกณฑ์การผ่าน

หน้าเว็บแสดงคำทักทายที่ตรงกับเวลาจริง แสดงวันในสัปดาห์เป็นภาษาไทยถูกต้อง ทดสอบโดยเปลี่ยน $hour เป็นค่าต่าง ๆ ชั่วคราว เพื่อตรวจสอบว่าทุกเงื่อนไขทำงานได้

คำแนะนำ

ใช้ date_default_timezone_set('Asia/Bangkok'); ที่บรรทัดแรกของไฟล์ เพื่อให้ date() แสดงเวลาไทยอย่างถูกต้อง (UTC+7) ไม่เช่นนั้นเวลาอาจผิดได้

5
ใช้คำสั่ง include แยก Header และ Footer

ปรับโครงสร้างโปรเจกต์ให้ใช้ include แยก Header และ Footer ออกจากหน้าหลัก

  • สร้างโฟลเดอร์ mysite/ ใน htdocs
  • สร้าง mysite/includes/header.php: DOCTYPE, html, head, body เปิด, header tag พร้อม nav
  • สร้าง mysite/includes/footer.php: ปิด main, footer tag, ปิด body, html
  • สร้าง mysite/index.php: ตั้งค่า $pageTitle แล้ว require_once header.php, เขียนเนื้อหา, require_once footer.php
  • สร้าง mysite/about.php ในรูปแบบเดียวกัน แต่เนื้อหาต่างกัน
เกณฑ์การผ่าน

ทั้ง index.php และ about.php แสดง Header และ Footer เหมือนกัน โดยไม่มีโค้ดซ้ำกัน แก้ไขที่ header.php เพียงที่เดียวแล้วทั้งสองหน้าเปลี่ยนพร้อมกัน

คำแนะนำ

ใช้ require_once __DIR__ . '/includes/header.php'; แทน path แบบ relative เพราะ __DIR__ คือ directory ของไฟล์ปัจจุบัน ทำให้ include ทำงานถูกต้องไม่ว่าจะเรียกจากที่ใด
